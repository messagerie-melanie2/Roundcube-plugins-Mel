Mode d'emploi (Courrielleur Mél)

- Cliquez sur "Télécharger la signature" pour enregistrer le fichier signature

- Dans le Courrielleur, cliquez sur "Outils", puis "Paramètres des comptes"

- Sélectionnez votre compte

- Si décochée, cochez l'option "Apposer la signature à partir d'un fichier"

- Cliquez sur le bouton "Choisir" puis sélectionnez le fichier signature précédemment téléchargé

- Cliquez sur OK

Votre signature est en place. Faites un test rapide en essayant de rédiger un nouveau courriel.

Mode d'emploi

- Cliquez sur "Copier la signature" pour copier la signature générée

- Cliquez sur "Outils", puis "Paramètres des comptes"

- Sélectionnez votre compte

- IMPORTANT : Si cochée, décochez l'option "Apposer la signature à partir d'un fichier"

- Dans la partie "Texte de signature", cochez "Utiliser HTML"

- Collez le code de la signature dans le cadre sous "Text de signature" (Ctrl + C)

- Cliquez sur OK

Votre signature est en place. Faites un test rapide en essayant de rédiger un nouveau courriel.
