Mode d'emploi (Courrielleur Mél)

- Cliquez sur "Copier la signature" pour copier le code généré

- Dans le Courrielleur, cliquez sur "Outils", puis "Paramètres des comptes"

- Sélectionner son compte

- Si cochée, décocher l'option "Apposer la signature à partir d'un fichier"

- Texte de signature : cocher "Utiliser HTML"

- Coller le code généré dans la zone "Texte de signature"

- Cliquer sur OK

Votre signature est en place. Faites un test rapide en essayant de rédiger un nouveau courriel.
